# RDP

Tags: #RDP #Comandos 

```bash
❯ remmina                            # Para podernos conectar a la maquina victima por RDP de forma grafica (Se hace con usuario no privilegiado en nuestra maquina)
	Ahi le pasamos la IP
	Aceptamos el certificado
	Colocamos el usuario y password validos
	Dominio: WORKGROUP (Ya que pertenece al grupo Administrator) / netmon (Dominio de la maquina victima)
	Aceptar
Y nos conecta por RDP
```

```bash
❯ hydra -L <user.list> -P <password.list> rdp://<IP>                   # Para ver si ese usuario y passwd son validos
❯ xfreerdp /v:<IP> /u:<Username> /p:<Passwd>                           # Para conectarnos por RDP
```
