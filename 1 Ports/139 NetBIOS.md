# NetBIOS

Tags: #NetBIOS

## Comandos Windows

```bash 
❯ nbtstat -c                 # Obtener el contenido del cache, tabla de nombres del NetBIOS, IPs
❯ nbtstat -a <IP>            # Obtener el nombre del NtBIOS de la maquina remota    
```

```bash 
❯ net view \\<IP> /ALL       # Muestra los directorios compartidos 
```
