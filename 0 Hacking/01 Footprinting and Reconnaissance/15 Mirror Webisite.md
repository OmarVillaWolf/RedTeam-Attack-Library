# HTTrack Web Site Copier 

Tags: #Hacking #Footprinting #Website #MirrorWebsite 

```bash 
# Windows tool 
❯ WinHTTrack Web Site Copier     # Analizar un sitio web copiandolo localmente a la maquina de atacante

Nota: 
	1. Abrir la tool en Windows 
	2. Agregar nombre del proyecto 
	3. Agregar las URLs a analizar en 'Web Addresses'
	4. En 'Accion' seleccionar 'Download web site'
	5. En 'Set Options' pestaña 'Scan Rules' seleccionar todos los tipos de archivos  
	6. Extra: En 'Set Options' pestaña 'Spider' seleccionar 'no robots.txt rules' y asi evitar ser bloqueados 
	7. Seleccionar 'Please adjust connection parameters'
	8. Finalizar 


# Más tools
❯ https://www.cyotek.com
```
