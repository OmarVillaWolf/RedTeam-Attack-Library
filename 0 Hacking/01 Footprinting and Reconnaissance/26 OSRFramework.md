# OSRFramework 

Tags: #Hacking #Footprinting 

```bash 
# Kali tool 
❯ domainfy -n domain -t all         # Busca los difernetes TLD del dominio y muestras las IPs

	# n = Nombre del dominio 
	# t = Tipos de 'TLD = top-level domain' y es todo lo que va despues del punto final del dominio, por ejemplo 'Google.com' donde el 'TLD = .com' 
```

```bash 
# Kali tool 
❯ searchfy -q "target username or profile name"   # Muestra los detalles de las redes sociales como Github, Instagram y Keyserverubuntu
```

```bash 
# Más tools
❯ usify        # Recolecta registros de cuentas dando un nombre de usuario
❯ mailfy       # Recolecta info de cuentas de emails
❯ phonefy      # Checa la existencia de series de telefonos 
❯ entify       # Extrae entidades usando expresiones regulares desde URLs
```
