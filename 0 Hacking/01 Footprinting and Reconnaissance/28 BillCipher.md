# BillCipher 

Tags: #Hacking #Footprinting 

```bash 
# Kali tool 
❯ python3 billcypher.py 
	❯ website           # Se usará un sitio web 
	❯ domain.com        # Ingresar el dominio
	❯ 1                 # Escoger la opción 1 que es 'DNS Lookup'
	❯ yes               # Para continuar 
	
	❯ 3                 # Escoger la opción 3 que es 'GeoIP Lookup' para mostrar las coordenadas de un dominio 
	❯ 4                 # Escoger la opción 4 que es 'Subnet Lookup'
	❯ 6                 # Escoger la opción 6 que es 'Page Links' 
	❯ 8                 # Escoger la opción 8 que es 'HTTP Headers'
	❯ 9                 # Escoger la opción 9 que es 'Host Finder'
```
