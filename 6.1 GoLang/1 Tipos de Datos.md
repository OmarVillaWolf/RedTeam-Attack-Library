# Tipos de datos 

Tags: #GoLang 

```bash 
`var`          =        Define una nueva variable.
`const`        =        Define un valor constante que no cambia.
`type`         =        Define un nuevo tipo de datos.
`struct`       =        Define un nuevo tipo de datos estructurados que contiene múltiples variables.
`map`          =        Define un nuevo mapa o variable hash.
`interface`    =        Define una nueva interfaz.
```

## Funciones 

```bash 
func   = Esta palabra clave se utiliza para definir una nueva función.
return = Esto sale de una función, opcionalmente devolviendo valores.
```

## Gestión de paquetes 

```bash 
import   = Esta palabra clave se utiliza para importar un paquete externo en el paquete actual.
package  = Esto especifica a qué paquete pertenece un archivo en Go.
```
