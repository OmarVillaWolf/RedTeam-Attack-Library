# Comandos Kitty

Tags: #Kitty #Bspwm 

## Bspwm comandos de la ventana

```bash 
Win + Enter                    # Inicias una terminal 
Ctrl + Win + Alt               # Preselector
Ctrl + Win + Alt + Espacio     # Quitar los preselectores de cada ventana
Win + s                        # Conviertir a una ventana flotate
Win + Alt + Flecha(Arriba, Abajo, Izquierda, Derecha)        # Resize de la ventana
Win + f                        # Full screen
Win + t                        # Tamaño normal de la ventana
Win + Clic Derecho             # Resize de la pantalla con el ratón
Win + q                        # Salir de la sesión
Win + Shift + Flechas          # Mover la ventana que queramos hacia un lado
Win + Shift + f                # Abrir Firefox
Win + Shift + 2                # Mandar la ventana actual al segundo entorno de trabajo
Win + Shift + q                # Cerrar la sesión e ir a la pantalla de bloqueo
Win + w                        # Cerrar la ventana
Win + Alt + r                  # Recargar los cambios
Win + 2                        # Ir al segundo entorno de trabajo
Win + Esc                      # Aceptar los cambios hechos
Win + d                        # Abrir Rofi y puedes buscar ahí
Win + flechas                  # Cambiar de ventana
```

## Consola Kitty Linux

```bash 
Ctrl + Shift + Enter            # Abrir múltiples ventanas dentro de la misma terminal
Ctrl + Shift + r                # Modificar el tamaño de la Kitty, (t=taller, s=smaller, q=quit)
Ctrl + Shift + t                # Crear una nueva ventana Kitty
Ctrl + Shift + Alt + t          # Renombrar la ventana actual de trabajo
Ctrl + Shift + .                # Cambiar la nueva Kitty a la inquiera (.=Moverla hacia la izquierda, ,=moverla a la derecha)
Ctrl + Shift + Flecha(Left)     # Moverte entre las ventanas
Ctrl + Shift + w                # Cerrar la ventana de la Kitty
Ctrl + Flecha                   # Moverte entre consolas de la Kitty
Ctrl + Shift + z                # Hacer zoom a la ventana
Ctrl + f                        # Cambiar la ventanas de posición (f=forwards, b=backwards)
Ctrl + l                        # Organizar las diferentes ventanas (l=ele)
Ctrl + Alt + Ratón              # Seleccionar
F1 Copy a
F2 Paste a
F3 Copy b
F4 Paste b
Esc "Dos veces"                 # Coloca automáticamente sudo
```
