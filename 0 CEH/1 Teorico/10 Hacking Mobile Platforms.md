## Mobile Attack Vectors

```bash 
1. Malware 
	1. Virus
	2. Application modification 
	3. OS modification 
2. Data Exfiltration 
	1. Extracted from data streams and email
	2. Print screen and screen scraping
	3. Copy to USB key and loss of backup
3. Data Tampering 
	1. Modification by another application 
	2. Undetected tamper attempts
	3. Jailbroken device
4. Data Loss
	1. Application vulnerabilities 
	2. Unapproved physical access
	3. Loss of device 
```
